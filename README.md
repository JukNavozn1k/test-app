# test-app

Короткое описание проекта.


**Ссылка на сайт**

Версия приложения из main доступна по: https://juknavozn1k-test-app-main-elw1ko.streamlit.app/

**Требования**:
- Python 3.8+ (установлен в системе)
- Poetry (рекомендуется) — опционально можно запускать напрямую через Python

**Установка (Windows, PowerShell)**

1. Клонируйте репозиторий и перейдите в директорию проекта:

```powershell
git clone <repo-url>
cd test-app
```

2. (Рекомендуется) Установите зависимости через Poetry:

```powershell
pip install poetry
poetry install
```

3. Альтернатива — запустить напрямую через Python (если зависимости установлены вручную):

```powershell
python -m pip install -r requirements.txt  # если есть requirements.txt
python main.py
```

**Запуск**

- Через Poetry:

```powershell
poetry run python main.py
```

- Или запустить `run.bat` напрямую в PowerShell:

```powershell
.\run.bat
```
